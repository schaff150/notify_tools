🤖 Gemini AI Personality

This prompt is prepended before asking Gemini to write the notification message. Make JellyDad sound exactly how you want.

Gemini will generate a fun, personalized SMS announcement. Without a Gemini API key (see Settings), a default template message is used instead.

🔊 ElevenLabs TTS Audio

When configured, Gemini's script is converted to audio and sent as an MMS attachment via Nginx. Optional — falls back to text-only SMS if not set.

The audio file will be saved to /app/audio inside the container. Map this volume to a path your Nginx instance serves publicly so the link in the SMS is accessible.

Full public URL to the directory where audio files are served. Must be accessible from the internet for the audio link in SMS.

🧪 Test Arr Notification

Fire the full pipeline with a fake movie — Gemini writes a personalized script per recipient, ElevenLabs generates individual audio files, and SMS is sent to all configured recipients. May take 10–20 seconds.

📱 SMS Gateway for Android

Install the free SMS Gateway app on your Android phone. Open the app, tap Local server, and copy the credentials shown on the Home screen.

Local mode (recommended): Base URL is http://[phone-ip]:8080 — texts send from your own phone number over LAN.

Cloud mode: Use https://api.sms-gate.app — works away from home but routes through sms-gate.app cloud.

Shown in the app under Local server or use https://api.sms-gate.app for cloud mode.
